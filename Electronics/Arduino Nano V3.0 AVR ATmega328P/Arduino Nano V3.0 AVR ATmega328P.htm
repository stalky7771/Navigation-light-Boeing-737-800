    Arduino Nano V3.0 – маленькая, самодостаточная, разъемо-совместимая с макетками плата на микроконтроллере ATmega328. Она более-менее совпадает по функциональности с Arduino Duemilanove/Uno, но имеет другой форм-фактор. Arduino Nano недостает только разъема питания, а для подключения к ПК применяется Mini-B USB кабель (нет в комплекте). Реплика оригинальной платы, производство - Китай.

    Внимание! В версиях Arduino IDE выше 1.6.20 появилась опция выбора версии бутлоадера (по умолчанию - новый), поэтому, старые версии плат могут не реагировать на программирование по USB.

    Для подключения к компьютеру используется микросхема CH340G (ссылка на драйвер ниже).

    Arduino Nano может быть запитана от Mini-B USB разъема или внешнего источника питания 6-12В (пин "Vin") или 5В стабильного внешнего питания (пин "5V"). Питание автоматически переключается на источник с более высоким напряжением.

    Характеристики:

    • Микроконтроллер: ATmega328P
    • Тип корпуса: TQFP-32
    • Рабочее напряжение: 5В
    • Входное напряжение (рекомендованное): 7-12В
    • Цифровых входов/выходов: 14 (из которых 6 могут быть использованы как ШИМ)
    • Аналоговых входов: 8
    • Сила тока на входах/выходах: 40 мА
    • Сила тока для 3.3В выхода: 50 мА
    • Память: 32 кБ из которых 2кБ используется бутлоадером
    • SRAM: 2 кБ
    • EEPROM: 1 кБ
    • Частота: 16 МГц
